Welcome to CastFrame!

Uses:

CastFrame can be used as a way to share a website or files over a local network.

Known Issues:

CastFrame currently does not support folders in the CastFrame folder.

Basic Instructions:

Add what you would like to brodcast to the CastFrame folder. If it has a .html extention it will display as a website. If an index.html file is included, that will display first when someone visits you address; otherwise a map of the CastFrame folder will apear.

Click "Start Server" to turn your server on and click "Stop Server" to shut the server off.

Give people your IP address and port number displayd on your screen when the server is turned on, if you wish them to find your server more easily.

Dat Folder:

Add Items to the Dat folder that you want to run to brodcast to people with the link through aplications like CastFrame reciever that handle the dat protcol (a P2P protocol).

Warning:

Anything in the CastFrame folder when your server is on can be seen by anyone on the same network. Turning on your server could present you with onforseen security risks.
