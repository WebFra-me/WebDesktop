Welcome to WebDesktop!

Uses:

WebDesktop: find, build, and use great desktop applications easily.

Home Screen:

The home screen is seprated into two areas. The Apps area is ware applications you have installed will show up. The MyApps area is ware applications you make live.

The Bazaar:

The Bazaar is the marketplace ware you can install all sorts of goodies from the repo of your choise.

Warning: WebDesktop does not garentee the quality of the apps in the repository you have chosen and is not liable for anything in regards to them.

Develop:

This is ware you go to start writing your own apps.

Logo:

Opens up the folder that contains all Webdesktops user files.

Local Broadcasting webserver:

Known Issues:

WebDesktop's www folder currently does not support folders for use in the local webserver.

Basic Instructions:

Add what you would like to brodcast to the "www" folder. If it has a .html extention it will display as a website. If an index.html file is included, that will display first when someone visits you address; otherwise a map of the "www" folder will apear.

Give people your IP address and port number displayd on your screen when the server is turned on, if you wish them to find your server more easily.

Logging: WebServer traffic can be found by going to view in the top menue and going to developer tools.

Warning: Anything in the "www" folder when WebDesktop is open can be seen by anyone on the same network. This could present you with a security risks depending on what you have in your "www" folder.

Note: For more information please visit: https://github.com/WebFra-me/WebDesktop/wiki
